## MedNeuro — 프로젝트·기능 요약

MedNeuro는 **NIfTI 등 뇌 영상 데이터**를 불러와 **2D 슬라이스**와 **3D 볼륨**으로 시각화하고, 뇌종양 영역 분석·검토를 돕는 **웹 서비스**입니다.

### 주요 기능

- **NIfTI 업로드 및 검증**: 브라우저에서 뇌 영상 파일 업로드, 형식·크기 등 기본 검증 후 분석 파이프라인으로 전달
- **2D 슬라이스 뷰어**: 축 방향 전환, 슬라이스 인덱스 이동, NIfTI 헤더·차원 정보 표시 등 단일 워크스페이스에서 탐색
- **3D 볼륨 시각화**: 업로드·2D와 동일한 네비게이션으로 3D 전환, 볼륨 로딩·렌더링 및 회전·확대 등 조작
- **코멘트·협진 기록**: 슬라이스·볼륨 좌표와 연계한 코멘트 CRUD, 마커·진단 메모 등 재현 가능한 기록
- **인증·역할 분기**: 로그인 및 의사/일반 등 권한에 따른 화면·동선 분기
- **의료 데이터 조회**: 의료진 기준 환자·MRI 분석 목록 등 데이터 파이프라인

### 기술·구성 (요약)

- **Backend**: Spring Boot, MyBatis, Oracle Database  
- **Frontend**: JSP, JavaScript(jQuery/Ajax), Bootstrap — 클라이언트 일부는 React 기반 업로드·2D·3D·세션 설계와 연동  
- **영상 파이프라인**: NIfTI 디코딩·메타데이터·2D 타일·3D 메시/텍스처 준비는 서버/워커에서 수행하고 REST 등으로 클라이언트에 전달하는 구조  
- **대용량 처리**: `.nii` 등은 DB에는 메타·경로만 두고 바이너리는 스토리지에 두는 전제

---

### 1. 인증 비즈니스 로직 및 Ajax 통신 구현 (Backend & API)

- **비동기 ID 중복 체크**: @ResponseBody와 Ajax를 활용하여 페이지 새로고침 없이 실시간으로 아이디 중복 여부를 확인하는 기능을 구현했습니다.
- **사용자 권한 기반 분기(RBAC)**: 로그인 성공 시 세션에 사용자 유형(의사/일반)을 저장하고, 이에 따라 최적화된 메인 페이지로 리다이렉트하는 컨트롤러 로직을 설계했습니다.
- **세션 기반 로그아웃**: Ajax 요청을 통해 세션을 즉시 무효화(Invalidate) 처리하는 안전한 로그아웃 프로세스를 구축했습니다.

### 2. 데이터 무결성 및 예외 처리 (Data Integrity)

- **DB 제약 조건 기반 예외 설계**: 회원가입 시 발생할 수 있는 데이터 충돌(주민번호, 전화번호 중복 등)을 `DataIntegrityViolationException`으로 포괄하여 처리함으로써 DB 무결성을 유지하고 사용자에게 명확한 오류 피드백을 제공합니다.
- **사전 로직 검증**: 서비스 단에서 DB 조회 전 정규표현식(Regex)을 통한 형식 검증을 수행하여 비정상적인 데이터의 유입을 원천 차단했습니다.

### 3. 인증 인터페이스 및 클라이언트 로직 (UI/UX & JS)

- **회원가입 및 로그인 UI 설계**: JSP와 Bootstrap을 활용하여 의료 플랫폼에 적합한 신뢰감 있고 직관적인 인증 페이지(`login.jsp`, `register.jsp`)를 제작했습니다.
- **JavaScript 기반 유효성 검사**: 폼 제출(Submit) 전 프론트엔드 단에서 입력값의 누락 및 형식을 1차 검증하여 불필요한 서버 통신 비용을 절감했습니다.

### 4. 의료 데이터 관리 및 조회

- **의료진 전용 환자 리스트**: MyBatis 서브쿼리와 조인을 활용하여 담당 의사가 확인해야 할 최신 MRI 분석 리스트를 효율적으로 조회하는 데이터 파이프라인을 구축했습니다.

- **Database**: Oracle Database (Constraints/Sequence)
